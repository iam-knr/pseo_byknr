About the Plugin

What is PSEO PRO by KNR?

PSEO PRO is a WordPress plugin built for SEO professionals, digital marketers, and agencies who need to create large volumes of highly targeted pages from structured data — without writing a single line of code.

You bring the data (a CSV, a Google Sheet, a JSON feed, or a REST API). You build one template page. PSEO PRO does the rest — replacing {{placeholders}} with your data to generate hundreds or thousands of unique, SEO-ready pages instantly.

Each generated page gets its own title tag, meta description, canonical URL, robots tag and JSON-LD schema markup — all derived automatically from your data row.

Update your data source and PSEO PRO auto-regenerates only the changed pages, updates existing ones, and removes orphaned pages that no longer exist in the data.

data.csv → 500 pages

Under the Hood

How It Works

1

📥 Data Fetch

Connects to your data source — CSV URL, uploaded CSV, Google Sheet, JSON endpoint, or paginated REST API. Parses into a row array where each row becomes one page. No row limits.

CSV · Google Sheets · JSON · REST API
2

🔍 Row Hashing & Smart Diff

Each row is MD5-hashed and stored in wp_pseo_data_rows. On re-runs, only changed rows are re-processed. New rows → new pages. Changed rows → updated pages. Deleted rows → orphan pages removed automatically.

3

🧩 Template Engine

Your WordPress page is cloned per row. Every {{placeholder}} in the title, content, URL slug, meta title, meta description, and schema fields is replaced with that row's column value. Supports {{raw:col}}, {A|B|C} spintax, and [if:col=val]...[/if] conditionals.

4

🚀 Page Creation & SEO Injection

Pages created via wp_insert_post(). The SEO class hooks into wp_head at priority 1 to inject title tag, meta description, robots, canonical URL, and JSON-LD schema — all per-row. Compatible with Yoast SEO and Rank Math without conflicts.

JSON-LD Schema injected per page
5

🗺️ Sitemap & Auto-Sync

All generated pages appear in a custom XML sitemap at /pseo-sitemap.xml. WP Cron event pseo_cron_sync runs on your chosen schedule — re-fetches data and regenerates changed pages completely hands-free.

Template Engine

Syntax Reference

{{column}}

Inserts the HTML-escaped value of the CSV column named column.

{{city}} → "Bangalore"
{{raw:column}}

Inserts the raw, unescaped value — useful for HTML content stored in the CSV.

{{raw:description}} → <p>HTML</p>
{A|B|C}

Spintax — randomly picks one option per page generation for content variation.

{Best|Top|Leading} plumber → "Top plumber"
[if:col=val]...[/if]

Conditional block — shows content only when the column matches the value.

[if:city=Bangalore]Local offer[/if]
[if:col>0]...[/if]

Numeric comparison — supports =, !=, >, <, >=, <= operators.

[if:rating>=4]Top Rated[/if]
{{service}}/{{city}}

URL pattern — use placeholders in slugs for clean, keyword-rich URLs.

→ /plumbing/bangalore/

Step-by-Step Guide

From Zero to 1,000 Pages

Follow these 7 steps to launch your first programmatic SEO campaign.

1

Install & Activate the Plugin

Upload the plugin zip via WP Admin → Plugins → Add New → Upload Plugin. After activation, go to Settings → Permalinks → Save Changes to flush rewrite rules — skip this and your pages will 404.

💡 WP-CLI: wp plugin install pseo-pro-knr.zip --activate && wp rewrite flush
2

Prepare Your Data Source

Create your dataset. Row 1 must be column headers — these become your {{placeholder}} names.

  • CSV via URL — host a public .csv and paste its direct URL
  • Google Sheet — share as "Anyone with link → Viewer", copy the Sheet ID from the URL between /d/ and /edit
  • JSON URL — point to any JSON array, specify dot-notation data path
  • REST API — configure endpoint, pagination param, and optional Bearer token
💡 Google Sheets export URL: docs.google.com/spreadsheets/d/SHEET_ID/export?format=csv&gid=0
3

Build Your Template Page

Create a new WordPress page using Elementor, Divi, Gutenberg, or plain HTML. Write your content using {{column_name}} placeholders wherever you want data inserted.

  • Use {{city}} in H1, paragraphs, alt text, buttons
  • Use {{raw:description}} for HTML-heavy content columns
  • Use {Best|Top|Leading} spintax for natural variation
  • Use [if:state=Karnataka]...[/if] for regional content blocks
💡 Tip: Publish the template as Draft — it's the master template, not a live page.
4

Create a New PSEO Project

Go to PSEO PRO → New Project in your WP Admin sidebar. Fill in all project settings:

  • Project Name — internal label only
  • Generate as Post Type — Page, Post, or any custom post type
  • Content Template — select the draft page from Step 3
  • Data Source — pick source type, fill in URL / Sheet ID / API details
  • URL Pattern — e.g. {{service}}/{{city}}
  • SEO Title — e.g. {{service}} in {{city}} — From ₹{{price}}
  • Meta Description — 120–160 chars with placeholders
  • Schema Type — pick one of the 6 built-in types
  • Sync Interval — Manual / Hourly / Daily / Weekly
5

Preview Your Data First

Before generating, click 👁 Preview Data. This fetches your source and shows the first 5 rows, total row count, and all detected column names — so you can verify your placeholder names match exactly (case-sensitive).

⚠️ Common mistake: CSV has column City (capital C) but template uses {{city}} (lowercase). Always preview first and fix mismatches before generating.
6

Generate Pages

Click ⚡ Generate Pages Now. PSEO PRO fetches all rows, applies the template, and creates/updates WordPress pages in bulk. For 10,000+ pages, use WP-CLI to bypass PHP timeout limits:

  • wp pseo list — view all projects and their IDs
  • wp pseo generate --id=1 — generate project #1
  • wp pseo generate --all --delete-orphans — generate all + clean orphans
7

Submit Sitemap to Google Search Console

Your sitemap is live at https://yourdomain.com/pseo-sitemap.xml. Go to Google Search Console → Sitemaps and submit it. Google will start crawling your pages.

  • Enable Auto-Sync to keep pages updated as your data changes
  • Enable Delete Orphans to remove pages when rows are deleted
  • Monitor indexing in GSC under Coverage → Valid
💡 Pro Tip: Set Sync to Daily + Google Sheet as source. Update the sheet → PSEO PRO auto-regenerates overnight. Fully hands-free.
